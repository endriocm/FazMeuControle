# Fluxo Financeiro

Aplicação estática de controle financeiro pronta para a Vercel. Não usa Electron, API, banco de dados ou variáveis de ambiente.

## Recursos

- Entradas, saídas, categorias, cartões e compras;
- Indicadores e análises do período;
- Persistência no `localStorage` do navegador;
- Backup e importação em JSON.

## Deploy na Vercel

Defina `controle-financeiro-vercel` como **Root Directory** do projeto. Não há comando de build nem variáveis de ambiente a configurar.

## Dados

Os dados não saem do navegador. Use **Backup** antes de trocar de computador ou limpar os dados do navegador.
